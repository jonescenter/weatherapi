# Jones Center Data Portal

Welcome to the Jones Center at Ichauway data portal. This site provides documentation for the Jones Center Weather API, which exposes real-time weather station data for use in R and other analysis tools.

## Getting Started

The Weather API uses Microsoft Entra ID for authentication. You sign in with your existing `@jonesctr.org` account — no separate API key or account needed.

**Jones Center Tenant ID:** `d1f21831-a420-4b90-a1bd-be2d7f4025e1`

This value is shared across all Jones Center APIs and is referenced in the quickstart guide.

## Access & Support

Access to the Weather API is managed by IT. To request access contact the IT department at [support@jonesctr.org](mailto:support@jonesctr.org).
